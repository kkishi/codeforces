# 解法

数式で考えると難しいが、具体例で挙動を見ると答えが分かる。

l=1, r=10では

a=10: a%b={0}
a=9: a%b={0, 1}
a=8: a%b={0, 1, 2}
a=7: a%b={0, 1, 2, 3}
a=6: a%b={0, 1, 2, 3, 4}
a=5: a%b={0, 1, 2, 3, 4}
a=4: a%b={0, 1, 2, 3}
a=3: a%b={0, 1, 2}
a=2: a%b={0, 1}
a=1: a%b={0}

l=1, r=5では

a=5: a%b={0}
a=4: a%b={0, 1}
a=3: a%b={0, 1, 2}
a=2: a%b={0, 1}
a=1: a%b={0}

のように山なりになる。
